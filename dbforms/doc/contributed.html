This section contains contributed tools, extensions by individuals that are not (yet) part of the main distribution.

You are encouraged to send own contributions to be placed here.

  • Merger Stylesheet (contrib. by Mark Dimon)
    XSL stylesheet performs a merge of the config file generated by DevGui with a similar overaly file , this way you can force certain fields to be set , ie usfull for isKey in mysql and any other attributes / elements which are needed in the config but not actually generated.
